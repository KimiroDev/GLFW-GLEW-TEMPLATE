Initial configuration in root folder of project:
```
git clone https://github.com/glfw/glfw.git external/glfw
git clone https://github.com/nigels-com/glew.git external/glew
cmake -S . -B build/
```

Build:
```
cd build/
make
```
Run:(in build/)
```
./template    
```
